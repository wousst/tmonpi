tMonPi
======
A simple IoT system that monitor the temperature on Raspi.

Howto: Install
--------------
```
npm install --save jsonwebtoken mqtt redis shelljs date-time firebase-functions firebase-admin
```

Dependencies
------------
* Google IoT-Core
* Google Pub/Sub
* Google Cloud Functions
* Firebase Database
* MQTT protocols
* Google BigQuery
* NodeJS
